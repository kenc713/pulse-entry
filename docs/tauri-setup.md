# tauri-setup

- このドキュメントでは、軽量Windowsデスクトップアプリケーションの実装に適したTauriのセットアップ方法について説明する。

## Rustのインストール

https://learn.microsoft.com/ja-jp/windows/dev-environment/rust/setup?tabs=winget

## Microsoft C++ Build Toolsのインストール

https://v2.tauri.app/ja/start/prerequisites/#windows

## Tauriプロジェクトの作成

https://v2.tauri.app/ja/

- npmを使用して、Tauriプロジェクトを作成する。
- 対話型シェルに従って、プロジェクトの設定を行う。
- 設定完了後に`npm install`を実行して、必要な依存関係をインストールすると良い。
